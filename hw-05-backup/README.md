### Топология

Настроено **4 сервера**:
- `backup-server` — центральный сервер бэкапов
- `web-1`, `web-2`, `web-3` — клиенты, данные с которых копируются

Все серверы работают в единой сети, доступ по SSH настроен по ключам.

---

### Настройка резервного копирования

#### 1. Установка и настройка BorgBackup (или rsync)

На **backup-server** установлен BorgBackup:

```bash
sudo apt update
sudo apt install borgbackup -y
```
